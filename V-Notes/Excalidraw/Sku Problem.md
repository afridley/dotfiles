A Metrc Id must associate to a single sku. This means that we cannot use the Master sku since metrc items consider both flavor and cultivar.

We would be able to match a master sku but down the line matching to that sku would not provide us with the cultivar of the metrc item as we would lose it.

So we could instead use primary product sku's mapped to Metrc ID's.
